## 開發環境

- Windows Home 10 64bit
- VScode
- Python `3.7.0`
- numpy `1.21.6`
- requests `2.27.1`
- discord `1.7.3`

- yt-dlp `2022.5.18`

## 如何執行

> 預設 ***Windows Home 10 64bit*** 作業系統  

步驟 一：注意 ***.env*** 是否可用，若無法使用可參考使用 ***.json*** 。  

步驟 二：***.env***：記得填入 ***.env*** 需要的必要資料。  

步驟 二：執行機器人  

- 方法一：直接點兩下執行 `Bot.py`    
>
- 方法二：使用終端機

```bash
cd Python-discord-chatbot-fat_cat/tiny
```
```bash
python Bot.py
```
